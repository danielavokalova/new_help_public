# Help Release Checker

## Pre-merge checks

- Page follows structure and style guide
- Links and references are valid
- Navigation metadata is updated
- Build passes successfully
- PR includes preview URL

## Merge readiness

A docs change is merge-ready only when all checks above pass and required reviewers approved the PR.
