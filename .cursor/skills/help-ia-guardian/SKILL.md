# Help IA Guardian

## IA model

- getting-started
- configuration
- operations
- troubleshooting
- release-notes

## Rules

- Put each article in exactly one primary section.
- Prefer shallow hierarchy and descriptive slugs.
- If content overlaps, keep one canonical page and link from others.
- New articles are Markdown files: `content/docs/<section>/<slug>.md`, URL `/<section>/<slug>`.
