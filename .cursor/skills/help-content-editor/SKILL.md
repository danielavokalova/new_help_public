# Help Content Editor

## Use this skill when

- writing a new help page
- rewriting imported content
- improving clarity of existing procedures

## Required structure

1. Purpose
2. Prerequisites
3. Steps
4. Verification
5. Related articles

## Quality checklist

- Neutral and concise language
- One concept per section
- Numbered procedures
- No duplicate explanations across pages
