# Style Guide

## Voice and Tone

- Use neutral, direct language.
- Avoid marketing wording.
- Use one term consistently for the same concept.

## Page Template

Each article should contain:

1. Purpose
2. Prerequisites
3. Steps
4. Verification
5. Related articles

## Writing Rules

- Keep sentences short.
- Prefer active voice.
- Use numbered lists for procedures.
- Add screenshots only when text alone is unclear.
