# CEE Hot Icon — Overview

CEE Hot Icon is a lightweight web shortcut designed for Central & Eastern European clients. It installs as a Progressive Web App (PWA) with a branded icon directly on the Windows desktop or taskbar — no installation wizard required.

## What you get

- A white-square app icon featuring the GOL IBE compass logo
- One-click access to the GOL IBE booking interface
- Works in Chrome and Edge on Windows 10/11

## How to add it to the taskbar

1. Open **Google Chrome** or **Microsoft Edge**
2. Navigate to your GOL IBE web address
3. Look for the **Install** icon (⊕) in the browser address bar
4. Click **Install** — the app shortcut appears on your desktop
5. Drag the desktop shortcut to the **Windows taskbar**

That's it. The icon stays on your taskbar and opens GOL IBE in its own window, without browser chrome.

## Icon specification

| Property | Value |
|---|---|
| Background | White (#FFFFFF) |
| Shape | Square (no rounding at 192 × 192, slight rounding at 512 × 512) |
| Logo | GOL IBE compass — blue outer ring, red inner ring, white center |
| Sizes supplied | 192 × 192 px · 512 × 512 px |
