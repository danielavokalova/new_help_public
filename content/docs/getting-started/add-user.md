# Add a New User (Agent)

Admins and Supervisors can invite new agents to the agency account.

## Steps

1. Go to **Settings → Users → + Add User**.
2. Enter the agent's **email**, **first name**, and **last name**.
3. Select a **role**: Agent, Supervisor, or Admin.
4. Click **Send Invitation**.

The new agent receives an email invitation and sets their own password.
