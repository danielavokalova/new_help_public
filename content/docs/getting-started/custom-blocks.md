# Supporting Texts — Front-end Content

<!-- tags: supporting texts, footer, menu, front-end, content, custom -->

The **Supporting texts** section is where you manage custom text content that appears on your GOL IBE front-end: footer, navigation menu, terms and conditions, ticket itinerary template, and CSS styling.

## How to add or edit a supporting text

1. Log into the admin console.
2. Go to **Supporting texts → Supporting texts**.
3. Click **Add text** to create a new entry, or find an existing one and click **EDIT**.
4. Fill in:
   - **Name** — a recognisable label (e.g. "Footer")
   - **Text type** — choose the correct type (Footer, Menu 1, Menu 2, Ticket search – Travel conditions, etc.)
   - **Language** — the language your front-end is in
   - **Body** — insert plain text or HTML content, or provide an **Alternative URL** for external pages
5. Save changes.

> After saving, flush HTML caches so the updated content appears immediately on the front-end.

## What you can add

| Text type | Used for |
|-----------|----------|
| Footer | Contact details, links, legal text in the page footer |
| Menu 1, Menu 2, … | Navigation links in the front-end header |
| Ticket search – Travel conditions | Terms and conditions shown at checkout |
| CSS | Custom styling rules |
| Ticket itinerary | Layout of e-ticket PDFs sent to customers |
