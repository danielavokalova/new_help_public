# Custom Content Blocks

<!-- tags: content block, banner, notice, message, front-end, custom -->

Custom blocks let you add promotional banners, notices or informational messages to specific pages of the booking engine.

## Create a content block

1. Go to **Supporting Texts → Custom Blocks → New block**.
2. Choose the **position**:
   - Above search form
   - Below search results
   - On booking confirmation page
3. Enter your content (HTML supported).
4. Set **Active from / to** dates (optional).
5. Click **Save**.

## Use cases

- Promote a sale: "Summer fares from €99!"
- Maintenance notice: "Scheduled downtime on Saturday 02:00–04:00"
- Policy update: "New baggage rules effective 1 May"
