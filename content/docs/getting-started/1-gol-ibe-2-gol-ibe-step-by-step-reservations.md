# Reservations

<!-- tags: reservations, bookings, create, modify, cancel, ticket, export -->

The **Reservations** section is where all flight bookings are managed — whether created by customers on your front-end or by agents directly in the admin console.

## What you can do here

| Action | Description |
|--------|-------------|
| **Create** | New air reservations with passenger details |
| **Modify** | Change dates, names or services on existing bookings |
| **Cancel** | Cancel bookings and process refunds where applicable |
| **Issue tickets** | Issue manually or trigger auto-ticketing |
| **Export** | Download booking lists to CSV or Excel for reporting |
| **Block fake bookings** | Configure anti-abuse rules |

## Steps

1. Log into the GOL IBE admin console.
2. Go to **Reservations** in the left-hand menu.
3. Use the booking list to find an existing reservation, or click **New reservation** to create one.
4. Select the required action: modify, cancel, issue ticket, or export.

## Results

All bookings — whether created by customers or agents — are managed in one place. You have full visibility of each reservation's status, passengers, payments and ticketing.

## Tips

- Use **Export** to download booking lists for accounting or management reporting
- Enable **anti-abuse rules** to block automated fake booking submissions
- Tickets can be issued manually or configured to issue automatically after payment
