# How to rewrite our text for SEO

<!-- tags: SEO, search engine optimisation, meta description, keywords, text, front-end -->

GOL IBE provides default text for the search form and booking pages. You can rewrite this text to improve your search engine rankings and better match your brand voice.

## Where to change SEO text

### Meta description and keywords

1. Log into the GOL IBE admin console.
2. Go to **Dealers → Dealers** → your dealer → **Front-end settings**.
3. Find fields **Meta description** and **Meta keywords**.
4. Write a 150–160 character description of your booking service.
5. Click **Save** and flush HTML caches.

### Page content text (headings, labels)

1. Go to **Supporting texts → Supporting texts**.
2. Find the text entry for the page you want to change (e.g. `Homepage heading`).
3. Click **Edit**.
4. Rewrite the text in the **Body** field.
5. Click **Save** and flush HTML caches.

## SEO text tips

| Element | Recommendation |
|---------|---------------|
| Meta description | 150–160 chars, include main keyword naturally |
| H1 heading | One per page, describe the page topic clearly |
| Labels | Use words customers search for (e.g. "cheap flights" over "fare search") |
| Alt text | Describe images for accessibility and image search |

## Tips

- Focus on what makes your agency unique — price, routes, service
- Avoid duplicate content across multiple pages
- Flush HTML caches after every change so search engine crawlers see the updated text
- Check results in Google Search Console after a few weeks to see if rankings improve
