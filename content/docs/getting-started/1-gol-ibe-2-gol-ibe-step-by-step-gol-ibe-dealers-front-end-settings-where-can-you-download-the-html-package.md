# Where can you download the HTML package?

<!-- tags: HTML package, download, front-end, deployment, files, dealer -->

GOL IBE provides a downloadable HTML package that contains the template files for your booking engine front-end. You can use this package to host GOL IBE on your own server or integrate it into your existing website.

## How to download the HTML package

1. Log into the GOL IBE admin console.
2. Go to **Dealers → Dealers** and open your dealer record.
3. Click **Front-end settings**.
4. Look for the **HTML package** section or button.
5. Click **Download HTML package**.

The package is a ZIP archive containing the complete front-end template.

## What's in the package

| File / folder | Contents |
|--------------|----------|
| `index.html` | Main entry point |
| `css/` | Stylesheets including customizable color variables |
| `js/` | JavaScript logic for the booking engine |
| `images/` | Placeholder images and icons |
| `fonts/` | Web font files |

## How to deploy

1. Unzip the package on your web server or local environment.
2. Update the configuration file with your GOL IBE API endpoint and dealer token.
3. Upload the files to your hosting environment.
4. Test the booking engine on the front-end URL.

## Tips

- Download a fresh package after major GOL IBE updates to get the latest front-end improvements
- Keep a copy of your customized CSS so you can reapply changes after updating the package
- Contact CEE Systems support if the package download button is not visible in your console
