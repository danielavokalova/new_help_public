# How about section Blocked e-mails?

 

***

* Log into the admin console.
* Go to the section **Code list – Blocked e-mails.**
* There you simply just add an e-mail you would like to get rid off.
* Click at button **"Add".**
* Insert e-mail you would like to block from spamming you.
* Save the settings.
