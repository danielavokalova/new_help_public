# Booking Confirmation Notifications

<!-- tags: confirmation, email, booking, template, notification, client -->

GOL IBE sends two separate notifications when a booking is created — one to the client and one to the agency. This two-notification approach is intentional and protects against abuse.

## What gets sent and to whom

| Notification | Recipient | Contains PNR? |
|---|---|---|
| **Air tickets – Booking created** (without attachment and PNR) | Customer | No |
| **Air tickets – Booking created** (with attachment and PNR) | Agency and dealers | Yes, with ticket itinerary attached |

The customer receives a simple confirmation so they know the booking was submitted. The full details go only to you (the agency).

## How to edit a notification template

1. Log into the admin console.
2. Go to **Notifications → Notification templates**.
3. Find the template you want to change and click **EDIT**.
4. If you have multiple language versions, select the language first, then click **EDIT** again.
5. Edit the text, colours, or logo in the simple editor — or switch to **HTML** on the task bar for full HTML editing.
6. Save changes.

## Managing recipients (delivery)

Click **DELIVERY** next to any notification to control who receives it.
Set each recipient type (customer, dealer, agency, configurator) to **Yes** or **No**.
