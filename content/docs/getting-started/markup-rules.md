# Configure Markup Rules

Markup rules automatically add a service fee or margin to all (or specific) bookings.

## Steps

1. Go to **Settings → Prices → + New Rule**.
2. Choose rule type: **Fixed amount**, **Percentage**, or **Per segment**.
3. Set optional conditions: carrier, route, fare class, or booking channel.
4. Click **Save** and toggle the rule to **Active**.

## Tips

- Test with a dummy search before activating.
- Rules are applied in priority order — higher priority overrides lower.
- Use **Per segment** for multi-leg itineraries to avoid double-charging.
