# How to set up a working hours exception

<!-- tags: working hours, exception, holiday, public holiday, closure, schedule -->

Working hours **exceptions** let you override your regular schedule for specific dates — for example, to close on a public holiday or extend hours during a promotional period.

## How to add an exception

1. Log into the GOL IBE admin console.
2. Go to **Agency → Working hours**.
3. Click the **Exceptions** tab (or **Add exception**).
4. Set the **date** for the exception.
5. Choose:
   - **Closed** — booking engine is completely closed on this date
   - **Custom hours** — set different opening and closing times for this day
6. Optionally add a note (internal only).
7. Click **Save**.

## Examples

| Scenario | Configuration |
|----------|---------------|
| Christmas Day closed | Date: 25 Dec, Type: Closed |
| New Year's Eve early closing | Date: 31 Dec, Type: Custom hours 09:00–14:00 |
| Extended Sunday sale | Date: specific Sunday, Type: Custom hours 10:00–20:00 |

## Tips

- Exceptions take priority over the regular weekly schedule
- You can add multiple exceptions in advance for the whole year
- Exceptions only apply to the exact date — the regular schedule resumes the next day
- Customers see the exception hours (or closure message) just like regular working hours
