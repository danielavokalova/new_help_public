# Users

<!-- tags: users, agents, roles, permissions, passwords, access -->

The **Users** section lets you manage all staff accounts that have access to the GOL IBE admin console. Each user gets their own login credentials and a role that defines what they can see and do.

## What you can do here

| Action | Description |
|--------|-------------|
| **Create** | New agent accounts with specific access rights |
| **Assign roles** | Admin, Agent, Read-only, or custom roles |
| **Reset passwords** | For agents who have been locked out |
| **Deactivate** | Accounts for staff who have left the agency |
| **Manage own account** | Update your profile and password |

## Steps

1. Log into the GOL IBE admin console.
2. Go to **Users** in the left-hand menu.
3. Click **New user** to create an account, or select an existing user to edit.
4. Assign the appropriate role and save — then share the login credentials with the user.

## Results

Only authorised staff can access your GOL IBE admin console, each with permissions appropriate to their role. Keeping accounts up to date ensures security and accountability.

## Tips

- Always mark the correct **Dealer** when creating a user so the system links them to your agency
- Assign the minimum role needed for each user's job function to limit access to sensitive settings
- Deactivate accounts promptly when staff leave — do not share login credentials
