# Basic Front-end Customisation

<!-- tags: front-end, customisation, design, dealer, logo, colours, branding -->

Each dealer can have a uniquely branded front-end. Basic customisation lets you change the logo, colour scheme, and layout settings without writing any code.

## How to access front-end settings

1. Log into the GOL IBE admin console.
2. Go to **Dealers** and select the dealer you want to customise.
3. Click **Edit → Front-end settings**.

## What you can customise

### Logo
- Upload your agency or dealer logo (recommended size: 200×60 px, PNG or SVG)
- The logo appears in the top navigation of the booking engine

### Colours
- **Primary colour** — used for buttons and key UI elements
- **Header background** — top navigation bar colour
- **Link colour** — hyperlink colour throughout the front-end

### Layout
- **Show/hide** specific sections (e.g. promotional banners, news feed)
- **Homepage layout** — configure what appears on the booking engine homepage
- **Search form position** — top or centre of the page

### Contact details
- Agency phone number displayed in the header
- Support email shown in the footer

## Apply changes

After saving your settings, click **Flush HTML caches** (under Code Lists) to ensure changes appear immediately for visitors.
