# Create a Customer Profile

<!-- tags: customer, create, passenger, profile, new -->

Saving a customer profile speeds up future bookings — passenger details are pre-filled automatically.

## Steps

1. Go to **Customers → New Customer**.
2. Enter passenger details:
   - Title, first name, last name
   - Date of birth
   - Nationality and passport number
   - Contact email and phone
3. Optionally add **frequent flyer numbers**.
4. Click **Save**.

## Using a saved profile

During booking, type the passenger name in the search field and select the saved profile. All details are filled in automatically.
