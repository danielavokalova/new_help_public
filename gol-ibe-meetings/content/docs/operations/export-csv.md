# Export bookings to CSV

Download a list of all bookings linked to meetings for use in external reporting tools.

## Steps

1. Go to **Reports → Export bookings**.
2. Set the **date range** and any filters (meeting, cost centre, attendee).
3. Click **Export to CSV** or **Export to Excel**.

## Included columns

- Meeting title, dates, location
- Attendee name and email
- Booking type (flight / hotel / transfer)
- Carrier / supplier, route, dates
- Total cost and currency
- Booking status
