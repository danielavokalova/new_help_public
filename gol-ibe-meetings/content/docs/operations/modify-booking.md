# Modify a travel booking

Change flight or hotel details for an attendee after booking.

## Steps

1. Open the meeting and go to the **Travel** tab.
2. Find the booking you want to change and click **Modify**.
3. Select **Change flight** or **Change hotel**.
4. Search for alternatives and select the new option.
5. Confirm the change.

The attendee receives an updated booking confirmation email.

> Some fare classes are non-changeable. If the **Modify** button is greyed out, the ticket must be cancelled and rebooked.
