# Cost centre breakdown report

View meeting and travel spending by cost centre to monitor budget usage.

## Steps

1. Go to **Reports → Cost centres**.
2. Select one or more cost centres.
3. Set the date range and click **Run report**.

## Report columns

| Column | Description |
|---|---|
| Cost centre | Code and description |
| Meetings | Number of meetings charged |
| Attendees | Total attendee-trips |
| Total spend | Sum of all travel costs |
| Budget | Configured annual budget |
| % used | Spend as a percentage of budget |
