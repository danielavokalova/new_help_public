# Meeting summary report

The meeting summary report gives an overview of all meetings in a selected period.

## Steps

1. Go to **Reports → Meeting summary**.
2. Set the **date range** (start and end date).
3. Optionally filter by **status**, **organiser**, or **cost centre**.
4. Click **Run report**.

## Columns

| Column | Description |
|---|---|
| Meeting title | Name of the meeting |
| Dates | Start and end date |
| Location | City / venue |
| Attendees | Total number of attendees |
| Status | Current status |
| Total spend | Sum of all travel costs |

## Export

Click **Export to CSV** or **Export to Excel** to download the results.
