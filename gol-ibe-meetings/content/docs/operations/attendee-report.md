# Attendee report

See which meetings an individual has attended and their associated travel costs.

## Steps

1. Go to **Reports → Attendees**.
2. Search for the attendee by name or email.
3. Set the date range and click **Run report**.

The report shows all meetings the attendee participated in, with travel costs and booking statuses.
