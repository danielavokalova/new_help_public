# Travel spend report

Analyse total travel spending filtered by period, cost centre, or department.

## Steps

1. Go to **Reports → Travel spend**.
2. Set the **date range**.
3. Filter by **cost centre**, **meeting type**, or **carrier** as needed.
4. Click **Run report**.

The report shows total spend, broken down by flight, hotel, and transfer.
