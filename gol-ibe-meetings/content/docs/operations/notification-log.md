# Notification delivery log

The delivery log records every email sent by the system so you can diagnose failed or missing notifications.

## Access the log

Go to **Settings → Notifications → Delivery log**.

## Log columns

| Column | Description |
|---|---|
| Date/time | When the email was sent |
| Recipient | Recipient's email address |
| Template | Which template was used |
| Meeting | The related meeting (if any) |
| Status | Delivered, Bounced, or Failed |

## Resend a notification

1. Find the failed notification in the log.
2. Click **Resend**.
3. Confirm in the dialog.
