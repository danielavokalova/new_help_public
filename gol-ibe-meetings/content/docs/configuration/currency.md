# Currency settings

Set the default currency used for meeting budgets, service fees, and reports.

## Steps

1. Go to **Settings → Currency**.
2. Select the **default currency** from the dropdown.
3. If you work with multiple currencies, enable **Multi-currency mode**.
4. Under **Exchange rates**, choose:
   - **Automatic** — rates fetched daily from an external provider.
   - **Manual** — enter rates yourself.
5. Click **Save**.
