# Auto-approval conditions

Some meeting requests can be approved automatically without manual intervention.

## Configure auto-approval

1. Go to **Settings → Approvers → Auto-approval**.
2. Click **Add rule**.
3. Set the conditions that must all be met:
   - **Maximum budget** — auto-approve if total travel cost is below this amount.
   - **Meeting type** — e.g. only auto-approve Internal training meetings.
   - **Advance notice** — e.g. only auto-approve if submitted at least 14 days in advance.
4. Click **Save**.

When all conditions are met, the request is automatically set to **Approved** and the organiser is notified immediately.
