# Working hours

Define when your support team is available. Meeting requests submitted outside these hours are queued and processed at the start of the next working day.

## Steps

1. Go to **Settings → Working hours**.
2. For each day of the week, enable the toggle and set the **start** and **end** times.
3. Set your **time zone**.
4. Click **Save**.

## Exceptions

To add a one-off exception (e.g. public holiday):
1. Click **Add exception**.
2. Set the date and mark it as **Closed** or enter custom hours.
3. Click **Save**.
