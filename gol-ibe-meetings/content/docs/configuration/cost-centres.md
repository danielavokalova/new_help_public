# Cost centres and budgets

Cost centres let you categorise meeting spending by department, project, or business unit.

## Create a cost centre

1. Go to **Settings → Cost centres** and click **Add cost centre**.
2. Enter the **code** (e.g. "MKTG-2026") and a **description**.
3. Optionally set an **annual budget** — the system will warn when 80% is reached.
4. Assign one or more **approvers** to the cost centre.
5. Click **Save**.

## Assigning users to cost centres

1. Go to **Settings → Users** and open the user's profile.
2. Under **Cost centres**, click **Add** and select from the list.
3. Click **Save**.

Organisers can only create meetings against cost centres they are assigned to.
