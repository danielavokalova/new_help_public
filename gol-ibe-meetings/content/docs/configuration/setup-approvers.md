# Set up approvers

Approvers are assigned to cost centres and are notified when a meeting request is submitted.

## Steps

1. Go to **Settings → Approvers** and click **Add approver rule**.
2. Select the **cost centre** this rule applies to.
3. Select the **user** who will act as approver.
4. Set the **approval level**:
   - **Level 1** — first to receive the request.
   - **Level 2** — notified after Level 1 approves (optional).
5. Set a **budget threshold** — requests above this amount require Level 2 approval.
6. Click **Save**.

## Tip

Assign a **backup approver** so requests are not blocked when the primary approver is away.
