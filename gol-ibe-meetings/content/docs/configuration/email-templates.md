# Customize email templates

All system emails use templates that can be edited to match your brand.

## Available templates

| Template | When sent |
|---|---|
| Meeting invitation | When a meeting is approved and attendees are notified |
| Approval request | When a request is submitted to the approver |
| Approval decision | When the approver approves or rejects |
| Booking confirmation | When travel is booked for an attendee |
| Booking cancellation | When travel is cancelled |
| Password reset | When a user requests a new password |

## Edit a template

1. Go to **Settings → Notifications → Email templates**.
2. Click the template you want to edit.
3. Use the rich-text editor to modify the subject line and body.
4. Use **merge fields** (e.g. `{{attendee_name}}`, `{{meeting_title}}`, `{{approval_link}}`) to insert dynamic content.
5. Click **Save** and optionally **Send test email** to preview.
