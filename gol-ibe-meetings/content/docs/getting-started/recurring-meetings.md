# Recurring meetings

Use recurring meetings to create a series of identical meeting requests without repeating the same setup each time.

## How to create a recurring series

1. Create a new meeting request as usual.
2. In the **Recurrence** section, select a frequency:
   - **Weekly** — same day each week
   - **Monthly** — same date each month
   - **Custom** — specify individual dates
3. Set the **End date** for the series.
4. Click **Submit for approval** — all instances in the series are submitted together.

## Managing a series

- To edit a **single instance**, open that meeting and click **Edit this occurrence**.
- To edit **all future instances**, click **Edit this and all following**.
- Cancelling a single occurrence does not affect the rest of the series.
