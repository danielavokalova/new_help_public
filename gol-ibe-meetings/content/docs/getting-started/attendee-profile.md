# Attendee profile overview

Each attendee has a profile that centralises their contact details, travel preferences, and travel documents.

## Profile fields

| Field | Description |
|---|---|
| First / last name | Full legal name as it appears on travel documents |
| Email | Used for all notifications |
| Phone | Optional; used for urgent notifications |
| Department | Linked to cost centre reporting |
| Travel preferences | Seat, meal, loyalty numbers |
| Passport | Number, expiry date, nationality |

## Accessing a profile

1. Go to **Attendees** in the navigation.
2. Search by name or email.
3. Click the attendee's name to open the full profile.
