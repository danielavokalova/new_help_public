# Roles and permissions

GOL IBE Meetings has four built-in roles.

## Role overview

| Role | Create meetings | Approve requests | Book travel | Manage settings |
|---|---|---|---|---|
| **Organiser** | Yes | No | No | No |
| **Approver** | No | Yes | No | No |
| **Travel agent** | No | No | Yes | No |
| **Admin** | Yes | Yes | Yes | Yes |

## Assigning roles

Roles are set when you [add a new user](/portal/getting-started/add-user) or can be changed later under **Settings → Users → Edit user**.
