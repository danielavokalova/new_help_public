# Add a new user

Create a new account for an organiser, approver, or travel agent.

## Steps

1. Go to **Settings → Users** and click **Add user**.
2. Enter the user's **first name**, **last name**, and **email address**.
3. Select a **role**:
   - **Organiser** — can create and manage meetings.
   - **Approver** — can approve or reject meeting requests.
   - **Travel agent** — can book travel for approved meetings.
   - **Admin** — full access to all settings and data.
4. Assign the user to one or more **cost centres** if required.
5. Click **Save**.

The user receives a welcome email with a link to set their password.
