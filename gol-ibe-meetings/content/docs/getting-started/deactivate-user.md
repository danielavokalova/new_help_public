# Deactivate a user

Deactivating a user prevents them from logging in without deleting their history.

## Steps

1. Go to **Settings → Users**.
2. Click the user's name.
3. Click **Deactivate**.
4. Confirm in the dialog.

The user is immediately signed out and cannot log back in. Their past meetings and bookings remain visible in the system for reporting purposes.

## Reactivating a user

To restore access, open the user's profile and click **Reactivate**.
