# Edit a meeting request

You can edit a meeting request as long as it has not yet been **approved** or **cancelled**.

## Steps

1. Open **Meetings → My requests**.
2. Click the meeting you want to edit.
3. Click **Edit** in the top-right corner.
4. Update any fields — title, dates, location, cost centre, or notes.
5. Click **Save changes**.

> If the meeting is already in **Pending approval** status, saving your changes will reset it back to **Draft** and you will need to re-submit for approval.

## Restrictions

| Status | Can edit? |
|---|---|
| Draft | Yes |
| Pending approval | Yes — resets to Draft |
| Approved | No — contact your approver |
| Cancelled | No |
