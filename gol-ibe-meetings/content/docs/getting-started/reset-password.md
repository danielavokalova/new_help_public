# Reset a user's password

Admins can trigger a password reset for any user.

## Steps

1. Go to **Settings → Users**.
2. Find the user and click their name.
3. Click **Reset password**.
4. Confirm in the dialog.

The user receives an email with a secure link to set a new password. The link expires after 24 hours.

## Self-service reset

Users can also reset their own password from the login page by clicking **Forgot password?** and entering their email address.
