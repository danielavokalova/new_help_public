# Meeting request not submitted

The **Submit for approval** button is greyed out or the submission fails.

## Common causes and fixes

**Missing required fields**
: Open the meeting and look for fields highlighted in red. Fill in all required fields before submitting.

**No cost centre assigned**
: At least one cost centre must be selected. Go to **Meeting details → Cost centre** and pick one from the list. If the list is empty, ask your Admin to [set up cost centres](/portal/configuration/cost-centres).

**No approver configured for the cost centre**
: The system needs to know who to route the request to. Ask your Admin to [set up approvers](/portal/configuration/setup-approvers) for the selected cost centre.

**Session timeout**
: If you have been inactive for a while, your session may have expired. Save a draft first, then log back in and re-submit.
