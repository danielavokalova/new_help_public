# User cannot log in

A user reports they cannot access the GOL IBE Meetings console.

## Checklist

1. **Password reset** — the most common fix. Go to **Settings → Users**, open the user, and click **Reset password**.
2. **Account deactivated** — check whether the account status is **Inactive**. If so, click **Reactivate**.
3. **Wrong email address** — verify the user is entering the correct email. The email must match exactly what is stored in **Settings → Users**.
4. **Browser cache** — ask the user to clear their browser cache and cookies and try again.
5. **Account does not exist** — if the user is new, you may need to [add them](/portal/getting-started/add-user) first.
